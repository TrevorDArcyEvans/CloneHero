## Prerequisites
* git
* Java SDK
* [JavaFX SDK](https://openjfx.io/index.html) 
* maven

## Getting started

```bash
$ git clone https://github.com/TrevorDArcyEvans/CloneHero.git
$ cd CloneHero
$ mvn clean
$ mvn compile
$ mvn test
$ mvn package
```
